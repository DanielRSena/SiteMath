Sistemas Lineares



Antes de tudo, o que é um sistema Linear?

São um conjunto de equações (vamos tratar apenas com duas) que compartilham de incógnitas iguais (também serão duas nesse caso) e tem relação entre si, como no modelo abaixo:

2x + 1y = 3,5
1x + 2y = 2,5


Como resolver

O intutito desses problemas matemáticos é, justamente, encontrar os valores das incógnitas (x e y, nesse caso). As três formas escolhidas para chegar nesses valores são adição, substituição e comparação:

  1. Adição

    2. Tendo como modelo o sistema acima, vamos encontrar o valor das incógnitas separadamente, com início pela X:

    2x + 1y = 3,5
    1x + 2y = 2,5

    Para tal, precisamos anular a variável Y, o que será possível quando seus valores forem iguais, porém opostos entre si. Para chegarmos nisso, se faz necessário multiplicar toda a primeira equação pelo número que acompanha o Y da segunda e vice versa, da seguinte forma:

    2x + 1y = 3,5 (multiplica por 2)
    1x + 2y = 2,5 (multiplica por -1)

    4x + 2y = 7
    -1x -2y = -2,5

    Nessa etapa entra a "adição" desse método, pois literalmente, somaremos o que conseguimos (lembre-se de fazer isso apenas com valores que multiplicam as mesmas incógnitas), da forma que está acima:

    3x + 0y = 4,5
    3x = 4,5
    x = 4,5 / 3
    x = 1,5


    Agora que sabemos o valor de X, sobra desvendar o valor de Y. Para isso, utilizamos qualquer umas das euqações inicias do sistema (quis usar a primeira) e substituimos o valor que encontramos agora na variável correspondente e terminamos a conta. Logo, tem-se que:

    2x + 1y = 3,5
    2(1,5) + 1y = 3,5
    3 + y = 3,5
    y = 3,5 - 3
    y = 0,5


  2. Substituição

    3. Vamos usar como exemplo o sistema acima. Assim como no anterior, vamos descobrir primeiramente o valor de X, para só então encontrar o da Y.

    Na subtituição, isolamos uma das variáveis (X) de um sistema e subtituimos no segundo. EX:

    Nesse caso, vamos isolar o X da primeira equação

    2x + 1y = 3,5
    2x = 3,5 - 1y
    x = (3,5 - 1y) / 2

    Agora vamos considerar que o X da segunda equação equivale a (3,5 - 1y) / 2, que é igual ao X da primeira. Ficaria assim:

    1((3,5 - 1y) / 2) + 2y = 2,5
    1,75 - 0,5y + 2y = 2,5
    - 0,5y + 2y = 2,5 - 1,75
    1,5y = 0,75
    y = 0,75 / 1,5
    y = 0,5

    Agora com o valor de Y, podemos substituir qualquer uma das equações inciais para encontrar o valor de x. Faremos com a segunda dessa vez:

    1x + 2y = 2,5
    1x + 2(0,5) = 2,5
    1x + 1 = 2,5
    1x = 2,5 - 1
    x = 1,5

  3. Comparação

    4. O processo é identico ao anterior (substituição), porém agora faremos isso com as duas equações. Dessa vez isolaremos o Y:

    Primeira equação

    2x + 1y = 3,5
    1y = 3,5 - 2x
    y = 3,5 - 2x

    Segunda equação

    1x + 2y = 2,5
    2y = 2,5 - 1x
    y = (2,5 - 1x) / 2
    y = 1,25 - 0,5

    Com as duas igualadas, podemos apenas juntálas com o '=' e resolver para encontrar o valor único de X

    3,5 - 2x = 1,25 - 0,5x
    3,5 - 1,25 = 2x - 0,5x
    2,25 = 1,5x
    2,25 / 1,5 = x
    1,5 = x

    Subsutituindo na segunda equação temos:
    1x + 2y = 2,5
    1(1,5) + 2y = 2,5
    2y = 2,5 - 1,5
    y = 1 / 2
    y = 0,5


Assim como era esperado, todas as formas de resolver o sistema levam aos mesmos resultados (x = 1,5 e y = 0,5). Logo, use a estratégia que julgar melhor.
